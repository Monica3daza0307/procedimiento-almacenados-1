Loop

La tercer estructura repetitiva que nos suministra MySQL se llama 'loop'. Este tipo de estructura repetitiva encierra un bloque de instrucciones entre los comandos 'loop' y 'end loop'. Para finalizar el bloque encerrado por los comandos 'loop' y 'end loop' debemos ejecutar el comando 'leave' e indicar una etiqueta definida previa al comando 'loop'. La sintaxis es la siguiente: [etiqueta]:loop [instrucciones] end loop;
